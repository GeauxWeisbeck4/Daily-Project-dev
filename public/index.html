The Daily Project Dev

Welcome to the Daily Projects Dev website, where I blog about my daily projects and share daily ideas for web dev projects!

This website was inspired by Tiny Projects.

I originally was going to create a similar site to Tiny Projects, months ago. Things got busy, I forgot, I started my fifth Section 4 business sprint - whatever the excuse is, I just flat out never did it. This has caused a problem with many of the little projects I have started myself while coding the past year. I've got a lot of projects going, but there are so many that get left undone. I also want to track my projects more closely so I can put myself in a better position to monetize them.


I am excited about this project given that I had almost completely forgotten about Tiny Projects. I made this website by using Eleventy, a static site generator, andPaper CSS. I used 11ty because of how easy it is to use and you can customize the tempalting any which way you want. I decided to use Paper CSS as a CSS framework - I've always wanted to use it in a project and I thought it would give this one a cute, personable look.


Every idea I launch, I will initially try to make money off of it - if it doesn't work out, just drop it and move on. I will at least have the idea recorded if anyone else wants to try and implement the project better than me! Some days I may just have to do a little design project or something small, just due to time constraints, but an idea a day would be pretty awesome. I would have to be successful with so many ideas going. Well, I suppose I better get started so see you down the road in my blog!
